# Skills Map

This document tracks the Java skills this project is intended to teach.

## Core Java

- Classes and objects: application entry point, services, repositories.
- Records: immutable transaction data.
- Enums: transaction type and category.
- Interfaces: storage and export contracts.
- Collections: in-memory transaction lists.
- Generics: reusable repository and result structures where useful.
- Streams: filtering, grouping, and monthly summaries.
- Optional: safe lookup results.
- Exceptions: validation and import/export failures.

## Java Standard Library

- `BigDecimal`: accurate money calculations.
- `LocalDate`: transaction dates.
- `UUID`: transaction identifiers.
- `Path`, `Files`, and related APIs: file import/export.

## Testing

- JUnit 6 test structure.
- Assertions for records and services.
- Tests for filtering and summaries.
- Tests for CSV/JSON import and export.
- Tests for validation errors.

## Design Skills

- Clean package structure.
- Separation of concerns.
- Domain-first development.
- Refactoring checkpoints.
- CLI usability.
- GitHub-ready documentation.
